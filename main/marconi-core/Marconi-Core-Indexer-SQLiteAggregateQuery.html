Marconi.Core.Indexer.SQLiteAggregateQuery

Description

A SQLiteAggregateQuery provides a way to query the content of several SQLiteIndexers as if they were a single indexer.

Note that SQLiteAggregateQuery doesn't implement IsIndex@: it won't index anything, and won't pass any information to the indexers it relies on. It's sole purpose is to provide a unified access for queries.
