Runner

runIndexer :: (IsIndex (ExceptT IndexerError IO) a indexer, Closeable IO indexer, Point a ~ ChainPoint) => RunIndexerConfig a -> indexer a -> IO () #

Connect to the given socket to start a chain sync protocol and start indexing it with the given indexer.

If you want to start several indexers, use runIndexers.
